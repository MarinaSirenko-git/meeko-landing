# Meeko landing

Static landing page starter: HTML, layered CSS, and vanilla JS.

## Structure

- `index.html` — entry document
- `css/` — styles (import order defined in `styles.css`)
- `js/` — scripts
- `assets/images/` — raster images
- `assets/icons/` — SVGs or icon assets
- `assets/fonts/` — self-hosted font files

## Local preview

Open `index.html` in a browser, or serve the folder with any static file server.
