# AI Usage

AI was used in this project as a coding assistant and reviewer.
This project was built as a frontend implementation study based on a free Framer template.

## AI-assisted tasks

- Extracting and cleaning SVG markup from Framer-generated code.
- Helping write documentation and implementation notes.

## Manual work

The following parts were handled manually:

- Choosing the template and analyzing the original layout.
- Rebuilding the page structure in HTML.
- Writing and adjusting CSS layout.
- Matching spacing, typography, colors and responsive behavior.
- Testing the page in browser.
- Reviewing and editing all AI-assisted code before using it.
- Final implementation decisions.

AI was used to support the workflow, not to replace frontend implementation.
